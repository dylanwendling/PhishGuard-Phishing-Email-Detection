# PhishGuard-Phishing-Email-Detection
Project Overview PhishGuard is a tool designed to analyze .eml files and detect phishing attempts using a combination of heuristics and machine learning. 
We will use the PhishStorm dataset to train or fine-tune a lightweight ML model.
We will provide a working, desktop-responsive basic flask app. We will demonstrate its functionality by processing sample emails to show the calculated risk score, triggered rules, and the explanation panel.
